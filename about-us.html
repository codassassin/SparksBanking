About Us

SparksBanking is a website designed by Mr. Souvik Chatterjee to provide a Simple User Interface of a Basic Banking System for SparkBank. This website allows the user to view the Customers List of the SparkBank and view their details. In addition, the user can also perform money transfers between multiple users. This website also provides the feature to check the Transaction History by navigating to the "Transactions" Page.
